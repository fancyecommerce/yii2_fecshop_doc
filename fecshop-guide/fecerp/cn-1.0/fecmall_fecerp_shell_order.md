订单同步脚本
==========

> 将商城订单同步到erp，将erp订单状态同步到商城平台



### Fecmall订单同步脚本

脚本文件路径：

```
./addons/fecmall/fecerp/shell/order.sh
```

您安装fecerp后，可以执行这个脚本，进行清理

```
cd  ./addons/fecmall/fecerp/shell
sh order.sh

```


关于该脚本的更多介绍，可以参看：[ERP订单扩展其他平台](fecmall_fecerp_order_other_platform.md)
