### ERP网站配置



1.添加fecmall商城网站

您可以在这里添加商城网站

![](images/erp_5.jpg)


您需要在您的fecmall商城后台，添加一个后台用户，作为api用户，并且配置fecmall appapi入口，譬如

Api Domain：`http://fecshop.appapi.fancyecommerce.com`

Api User：`apiuser`

Api Password：`xxxx`
